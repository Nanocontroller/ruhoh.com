The Good Stuff

Universal Blog API

My main vision is to make it easier for all developers to share their knowledge.

Through my experience with Jekyll-Bootstrap, it become apparent that not everyone has enough ruby experience to get a healthy development environment installed on their machine - and ruby is not trivial to install!

Ruhoh’s Universal Blog API specification allows every developer, regardless of preferred language and platform to join the community.

Modular

Ruhoh uses a modular architecture. You can read more about how ruhoh works. Modularization decouples the main Ruhoh components, allowing you to customize or replace individual components as needed.

Templaters, converters, parsers, compilers, even the command-line tool can be extended or simply swapped out - Ruhoh is made to be customized.

Language Indifferent

In the year 2000, there will be Ruhoh modules written in tons of languages, simply because there’s no reason why there couldn’t be!

Developers can take advantage of the Universal Blog API to implement modules in any language in any build environment.
